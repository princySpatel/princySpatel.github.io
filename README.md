# Princy's Portfolio

A personal portfolio showcasing my journey as a Computer Science and Engineering student specializing in AI/ML. This space serves as a central hub for my technical projects and explorations.

View the live site here: [https://princyspatel.github.io](https://princyspatel.github.io)

## Tech Stack

* **Framework:** Astro
* **Styling:** Tailwind CSS
* **Deployment:** GitHub Pages & GitHub Actions

## Customizations

While the foundational UI was based on an open-source Astro template, I have heavily modified the underlying codebase. Key customizations include:
* Completely re-engineering the GitHub Actions CI/CD pipeline to support Node 22 and bypass strict package manager requirements.
* Resolving bleeding-edge bundler conflicts between Astro and Vite.
* Reconfiguring the static asset routing and metadata schemas to support custom project documentation.
